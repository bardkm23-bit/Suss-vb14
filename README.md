# SUSS VB14 Session Schedule

A live session schedule for the SUSS Venture Builder Programme Cohort 14.

## 🌐 Live Preview

**[https://bardkm23-bit.github.io/Suss-vb14/](https://bardkm23-bit.github.io/Suss-vb14/)**

> The page is automatically deployed to GitHub Pages on every push to `main`.
> To enable it the first time, go to **Settings → Pages** and set the source to **GitHub Actions**.

## Features

- Countdown timer to the next session (Singapore Time, SGT UTC+8)
- Live badge shown during an active session
- All sessions listed with past / up-next / live highlights
- Mobile-friendly layout
